# Bank Customer Churn Analysis

## Project Overview

Customer churn is one of the biggest challenges faced by banks and financial institutions. This project analyzes customer behavior to identify factors contributing to customer churn and provides actionable insights to improve customer retention.

The project uses MySQL for data storage, Python for data analysis, and Streamlit for interactive dashboard visualization.

---

##Objectives

* Analyze customer churn patterns
* Identify key factors leading to customer attrition
* Compare churn across customer segments
* Generate business insights and retention recommendations
* Visualize churn trends through an interactive dashboard

---

##Technology Stack

* **Database:** MySQL
* **Programming Language:** Python
* **Libraries:** Pandas, Plotly, MySQL Connector
* **Dashboard:** Streamlit
* **IDE:** Visual Studio Code

## Workflow

1. Import customer churn dataset (CSV)
2. Create MySQL database and customers table
3. Load data into MySQL using Python
4. Perform churn analysis using Pandas
5. Generate insights and business recommendations
6. Visualize results using Streamlit Dashboard
